# 面试知识
## 面试流程
每个面试时间大概1~2个小时
- 技术面试
- 负责人面试
- HR面试

## 面试准备
- JD描述分析
- 业务分析
- 技术栈准备
- 自我介绍

## 模拟一面
主要考察基础知识体系
- 面试技巧 页面布局类
- CSS盒模型 DOM事件类
- HTTP协议类 原型链类
- 面向对象类 通信类
- 前端安全类 前端算法类

## 模拟二面
主要考察业务的宽度
- 面试技巧
- 渲染机制类
- JS运行机制
- 页面性能
- 错误监控

## 模拟三面
- 面试技巧
- 业务能力
- 团队协作能力
- 带人能力

## 模拟终面
